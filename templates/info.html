{% endif %} {% if info == "exposure" %}

Select public or private exposure for your submission.

PUBLIC exposure means that your submission can be searched by anyone using the search tool, giving other users enough information for house matching as well as contact info.

PRIVATE exposure means that your submission can't be searched from other users. Regardless if your submission is private or not, as long as it is set as your primary one, you can use the search tool to look for potential matching houses.

{% endif %} {% if info == "primary_submission" %}

Select your submission to behave as your primary one or not.

PRIMARY set means that your submission can be used for smart house matching when you are using the search tool. In addition it is subject to adjust the ranges when you are using the Matching Tolerance slider.

Every user must have one primary submission.

By default, your first submission is set in such way.

If you wish to change your primary submission, then you should create a new submission or edit an existing one and set it as primary.

{% endif %} {% if info == "search" %}

The search tool can filter out all the Swappr houses that are publicly exposed. Matching is done based on your primary submitted house characteristics.

Use House Characteristics and Location filtering mechanisms to narrow down the search results as desired.

Use Matching Tolerance slider to extend the ranges of the matching algorithm.

Matching colors explanation

Green rows are very high matches.

Blue rows are average matches.

Yellow rows are very low matches.

Red rows do not match at all.

{% endif %} {% if info == "tolerance" %}
Tolerance explanation

Tolerance extends the ranges of the matching probability.

For instance, if your submitted house rental is €500 a tolerance of 10% will consider a house match that its rental is up to €550.

A 50% tolerance for the same example will match a house of €750 rental.

Tolerance affects all House Characteristics Filters.

Colors meaning

Green rows are very high matches.

Blue rows are average matches.

Yellow rows are very low matches.

Red rows do not match at all.
